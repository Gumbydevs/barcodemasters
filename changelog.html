Barcode Masters Change Log

Version 1.0.03 - Initial Release

  • First commit: Basic project setup with main page structure and initial files.

Version Updates & Bug Fixes

  • V6 of the DNA Extractor/Monster Generator released.
  • Version Number Updates for consistency across pages.
  • Fixed multiple debugging issues (login, Firebase connections, and database interactions).
  • Favicon updates: Added and fixed the favicon across multiple pages.
  • Layout fixes: Adjustments for responsiveness and consistency, including fixes for dark mode and login alignment.
  • Spelling and text content corrections in various sections, including tooltips and help pages.

Monster Management & Admin Tools

  • Monster Generation: New version added to pull data from the database for dynamic monster types.
  • Admin Tool for Monster Types: Tool created for managing monster types, with bug fixes and interface improvements.
  • DNA Extractor: Attempted fixes for better monster type generation and saving changes.
  • Monster Collection: Added buttons for renaming and deleting monsters. Updates to the collection page to improve clarity and presentation.
  • Monster Stats & Experience: Monsters are now added to the player's collection with improved stat tracking.
  • Fixed issues with monster collection updates, including proper naming and display of monster stats.

Training & Battle System

  • Training Page: Layout and functionality improvements, including clearer dropdowns and better alignment for training selections.
  • Training Fixes: Functionality for selecting training actions and results fixed.
  • Monster Stat Increases: Stats now adjust properly based on training progress.

Login & Profile

  • Login System: Firebase integration for user registration and login system improvements. New features to handle user IDs and database connections.
  • Profile Page: New styles and updates to the profile page layout and display. Dark mode fixes and better button alignment.
  • Login Debug: Various attempts and fixes to resolve login and user registration issues.

User Interface & Style

  • Dark Mode: Styling fixes and adjustments to dark mode support across pages.
  • Buttons & Links: Multiple updates to button styles, including alignment and functionality for the DNA Extractor tool and profile page.
  • Help Pages: New help text and disclaimer updates for clearer user guidance. A new help button added to the login page.

Miscellaneous

  • Content Updates: Several updates to content across various pages, including tooltips, help text, and instructions.
  • Debugging Tools: Debug toggles and version updates added to help with troubleshooting.
